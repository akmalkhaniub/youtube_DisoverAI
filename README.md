# YouTube Topic Intelligence Agent

## Features
- Automated daily YouTube searches
- Metadata collection & analysis
- Trend identification
- Streamlit dashboard

## Setup
```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

## Configuration
1. Create `.env` file
2. Set up PostgreSQL/MongoDB
3. Edit `config.yaml`
